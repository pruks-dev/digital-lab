## การทดลองที่ 6.1 การสร้าง Clock Divider

Clock บนบอร์ด DE10-Lite มีความถี่ 50 MHz ซึ่งมีความเร็วสูงเกินกว่าที่จะสังเกตการเปลี่ยนแปลงของ LED ได้

ในการทดลองนี้ ให้ออกแบบวงจร Clock Divider เพื่อลดความถี่ของ Clock ลงเหลือประมาณ 1 Hz

### ขั้นตอนการทดลอง

1. สร้างตัวแปรสำหรับนับจำนวน Clock
2. เขียน Process เพื่อนับ Clock
3. เมื่อครบจำนวนที่กำหนด ให้สลับสถานะของ LED
4. Compile และ Download ลงบอร์ด
5. ตรวจสอบว่า LED กระพริบประมาณ 1 ครั้งต่อวินาที
6. ใช้ Oscilloscope Probe สัญญาณเอาต์พุตของ Clock Divider (ขา FPGA ที่ต่อกับ LED) — วัด Frequency ที่ได้จริง เปรียบเทียบกับค่าที่ออกแบบไว้

#### ตารางที่ 6.0 การวัดสัญญาณ Clock Divider

| พารามิเตอร์ | ค่าที่ออกแบบ | ค่าที่วัดได้ |
|-------------|-------------|-------------|
| Frequency (Hz) | 1 | |
| Period (s) | 1 | |
| Duty Cycle (%) | 50 | |

### คำถามท้ายการทดลองที่ 6.1

1. เพราะเหตุใดจึงต้องใช้ Clock Divider
2. หากต้องการให้ LED กระพริบเร็วขึ้น ต้องแก้ไขส่วนใดของโปรแกรม

## การทดลองที่ 6.3 การสร้าง Mod-10 Counter

ในการทดลองนี้ ให้สร้าง Counter ที่นับเฉพาะ 0–9 แล้วกลับไปเริ่มที่ 0

### ขั้นตอนการทดลอง

1. แก้ไขเงื่อนไขการนับ
2. เมื่อ Counter เท่ากับ 9 ให้กลับเป็น 0
3. Compile โปรแกรม
4. Download ลงบอร์ด
5. ทดลองการทำงาน
6. ใช้ Oscilloscope Trigger ที่สัญญาณ Reset — จับภาพ Moment ที่ Counter นับถึง 9 แล้วรีเซ็ตกลับเป็น 0

#### ตารางที่ 6.2 ผลการทดลอง

| Count | ค่าแสดงผล |
|--------|-----------|
|0||
|1||
|2||
|...|...|
|9||
|10||

## การทดลองที่ 6.4 การแสดงผลบน Seven-Segment Display

ให้นำ Mod-10 Counter จากการทดลองที่ 6.3 มาเชื่อมต่อกับ Seven-Segment Decoder ที่สร้างไว้ใน Lab 4

### ขั้นตอนการทดลอง

1. นำโมดูล Seven-Segment Decoder กลับมาใช้งาน
2. เชื่อมต่อ Counter เข้ากับ Decoder
3. Compile โปรแกรม
4. Download ลงบอร์ด
5. ตรวจสอบการแสดงผล
6. ใช้ Oscilloscope Probe สัญญาณ Segment a–g — ตรวจสอบว่า Segment เปลี่ยนตามค่าที่นับได้ถูกต้อง

### คำถามท้ายการทดลองที่ 6.4

1. เพราะเหตุใดจึงสามารถนำ Seven-Segment Decoder จาก Lab 4 มาใช้ซ้ำได้
2. การแบ่งโปรแกรมออกเป็นหลายโมดูลมีข้อดีอย่างไร

## สรุปผลการทดลอง

อธิบายผลการทดลอง พร้อมวิเคราะห์ความถูกต้องของผลลัพธ์ และอธิบายสาเหตุของข้อผิดพลาด (ถ้ามี)

## คำถามท้ายใบงาน

1. Binary Counter แตกต่างจาก Mod-10 Counter อย่างไร
2. หากต้องการสร้าง Counter แบบนับถอยหลัง (Down Counter) จะต้องแก้ไขส่วนใดของโปรแกรม
3. Clock Divider มีความสำคัญอย่างไรในการออกแบบระบบดิจิทัล
4. เพราะเหตุใด Counter จึงจัดเป็นวงจร Sequential Logic
5. ยกตัวอย่างการประยุกต์ใช้งาน Counter ในระบบดิจิทัลอย่างน้อย 3 ตัวอย่าง
6. ความถี่ของ Bit 0, Bit 1, Bit 2 และ Bit 3 มีความสัมพันธ์กันอย่างไร (จากที่วัดด้วย Oscilloscope)
7. การใช้ Oscilloscope Trigger จับ Moment ที่ Counter Mod-10 รีเซ็ต ช่วยในการ Debug อย่างไร
