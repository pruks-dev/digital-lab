## แนวคิดสำคัญ: Pull-up และ Pull-down Resistor

ในการต่อ Push Button เข้ากับอินพุตของ IC ลอจิก หากปล่อยขาอินพุตไว้โดยไม่ต่อ VCC หรือ GND ขณะที่ไม่ได้กดปุ่ม ขานั้นจะอยู่ในสภาวะ **Floating** — ระดับแรงดันไม่แน่นอน อาจถูกตีความเป็น 0 หรือ 1 สลับไปมาเนื่องจากสัญญาณรบกวน ทำให้วงจรทำงานผิดปกติและกินกระแสมากขึ้น

**Pull-down Resistor** เป็นตัวต้านทานที่ต่อระหว่างอินพุตของ IC กับ GND (โดยทั่วไปใช้ค่า 10 kΩ) เมื่อไม่ได้กดปุ่ม ตัวต้านทานจะดึงแรงดันที่อินพุตลงสู่ GND (logic 0) เมื่อกดปุ่ม VCC (+5V) จะต่อตรงเข้ามา มีค่าความต้านทานต่ำกว่าทาง Pull-down มาก อินพุตจึงกลายเป็น logic 1

![วงจร Push Button พร้อม Pull-down Resistor ต่อกับ IC](images/lab-1/push-button-pull-down.svg)

- **ไม่กดปุ่ม:** อินพุตต่อลง GND ผ่าน 10 kΩ → logic 0
- **กดปุ่ม:** อินพุตต่อตรง VCC (+5V) → logic 1 (กระแสไหลผ่าน 10 kΩ ลง GND น้อยมาก)

**Pull-up Resistor** ทำงานกลับกัน คือต่อตัวต้านทานระหว่างอินพุตกับ VCC และต่อ Push Button ระหว่างอินพุตกับ GND ซึ่งการทดลองนี้จะใช้แบบ Pull-down เป็นหลัก

ในการทดลองต่อไปนี้ ทุกครั้งที่ใช้ Push Button เป็นอินพุตให้กับ IC ให้ต่อ Pull-down resistor 10 kΩ จากขาอินพุตของ IC ไปยัง GND เสมอ

## การทดลองที่ 1.2 การทดลองวงจร NOT Gate (Inverter)

NOT Gate เป็นลอจิกเกตพื้นฐานที่มีอินพุต 1 ตัวและเอาต์พุต 1 ตัว โดยให้ค่าตรงข้ามกับอินพุต

![สัญลักษณ์ NOT Gate](images/shared/not-gate.svg)

![IC 74HC04 Pinout](images/lab-1/ic-74hc04.svg)

### ขั้นตอนการทดลอง

1. ศึกษา Datasheet ของ IC 74HC04
2. ระบุขา VCC และ GND ของ IC
3. ต่อไฟเลี้ยง +5V และ GND ให้กับ IC
4. ต่อวงจร NOT Gate ด้วย IC 74HC04
   - ต่อ Pull-down resistor 10 kΩ จากขาอินพุตของ IC (ขา 1) ไปยัง GND
   - ต่อ Push Button ระหว่าง VCC (+5V) กับขาอินพุตของ IC (ขา 1)
   - ใช้ LED แสดงผลลัพธ์
5. ทดลองครบทุกกรณี
6. ใช้ Oscilloscope 2 ช่อง Probe อินพุต (ขา Input ของ IC) และเอาต์พุต (ขา Output ของ IC) พร้อมกัน — วัดระดับแรงดัน $V_{OH}$ และ $V_{OL}$ บันทึกในตารางที่ 1.2a
7. ใช้ Cursor ของ Oscilloscope วัดค่าทางเวลาขณะกดและปล่อย Push Button ดังนี้

   ![Timing Diagram: tr, tf, tPLH, tPHL](images/lab-1/timing-diagram.svg)

   - Rise Time ($t_r$): เวลาที่สัญญาณเอาต์พุตเปลี่ยนจาก 10% → 90% ของ $V_{OH}$
   - Fall Time ($t_f$): เวลาที่สัญญาณเอาต์พุตเปลี่ยนจาก 90% → 10% ของ $V_{OH}$
   - Propagation Delay $t_{PLH}$: เวลาจากจุด 50% ของสัญญาณอินพุตถึงจุด 50% ของสัญญาณเอาต์พุต (LOW→HIGH)
   - Propagation Delay $t_{PHL}$: เวลาจากจุด 50% ของสัญญาณอินพุตถึงจุด 50% ของสัญญาณเอาต์พุต (HIGH→LOW)
   บันทึกในตารางที่ 1.2b
8. บันทึกผลการทดลอง

#### ตารางที่ 1.2 Truth Table ของ NOT Gate

| A | Y |
|---|---|
| 0 |   |
| 1 |   |

#### ตารางที่ 1.2a การวัดระดับแรงดันของ NOT Gate (74HC04)

| อินพุต | แรงดันอินพุต (V) | แรงดันเอาต์พุต (V) | ระดับลอจิก |
|--------|------------------|-------------------|------------|
| 0      |                  |                   |            |
| 1      |                  |                   |            |

#### ตารางที่ 1.2b การวัดค่าทางเวลาของ NOT Gate (74HC04)

| พารามิเตอร์ | สัญลักษณ์ | ค่าที่วัดได้ | หน่วย |
|-------------|-----------|------------|-------|
| Rise Time | $t_r$ | | ns |
| Fall Time | $t_f$ | | ns |
| Propagation Delay (LOW→HIGH) | $t_{PLH}$ | | ns |
| Propagation Delay (HIGH→LOW) | $t_{PHL}$ | | ns |

### คำถามท้ายการทดลองที่ 1.2

1. NOT Gate มีประโยชน์อย่างไรในการออกแบบวงจรดิจิทัล
2. ถ้าต่อ NOT Gate ต่อกัน 2 ตัว จะได้ผลลัพธ์เป็นอย่างไร
3. Propagation Delay ($t_{PLH}$/$t_{PHL}$) ของ NOT Gate ส่งผลต่อการทำงานของวงจรดิจิทัลที่ความถี่สูงอย่างไร

## การทดลองที่ 1.4 การสร้างวงจร XOR Gate จาก AND, OR, NOT

XOR Gate สามารถสร้างได้จาก AND, OR, NOT Gate โดยใช้สมการ Boolean

$Y = (A \cdot \overline{B}) + (\overline{A} \cdot B)$

![วงจร XOR Gate จาก AND, OR, NOT](images/lab-1/xor-from-gates.svg)

### ขั้นตอนการทดลอง

1. วาด Logic Diagram จากสมการ XOR
2. ต่อวงจรบน Breadboard ด้วย IC 74HC08, 74HC32, 74HC04 โดยต่อ Pull-down resistor 10 kΩ จากขาอินพุต A และ B ไปยัง GND และใช้ Push Button ต่อกับ VCC (+5V) เป็นอินพุต
3. ทดลองครบทุกกรณี
4. ใช้ Oscilloscope Probe อินพุต A, B และเอาต์พุต Y — สังเกต Glitch ที่เอาต์พุตเมื่ออินพุตเปลี่ยนพร้อมกัน (เนื่องจาก Propagation Delay ไม่เท่ากันในแต่ละ Path)
5. ใช้ Function Generator ป้อนสัญญาณ Square Wave ความถี่ 1 kHz ที่อินพุต A และ Push Button ที่อินพุต B — สังเกตเอาต์พุตบน Oscilloscope
6. บันทึกผลการทดลอง

#### ตารางที่ 1.4 Truth Table ของ XOR Gate

| A | B | Y |
|---|---|---|
| 0 | 0 |   |
| 0 | 1 |   |
| 1 | 0 |   |
| 1 | 1 |   |

### คำถามท้ายการทดลองที่ 1.4

1. XOR Gate แตกต่างจาก OR Gate อย่างไร
2. สามารถสร้าง XOR Gate ด้วย NAND Gate เพียงอย่างเดียวได้หรือไม่
3. ถ้าต้องการสร้าง XOR Gate แบบ 3-input จะต้องใช้ XOR Gate กี่ตัว

## การทดลองที่ 1.5 การสร้างวงจร Half Adder

Half Adder เป็นวงจรที่ใช้บวกเลขฐานสองจำนวน 1 บิต โดยมีอินพุต 2 บิต และเอาต์พุต 2 ค่า ได้แก่

- Sum (S)
- Carry (C)

โดยมีสมการดังนี้

- $S = A \oplus B$
- $C = A \cdot B$

![บล็อกไดอะแกรม Half Adder](images/lab-1/half-adder.svg)

### ขั้นตอนการทดลอง

1. ใช้วงจร XOR Gate ที่สร้างในข้อ 1.4 และ IC 74HC08 (AND Gate) โดยให้ต่อ Pull-down resistor 10 kΩ จากขาอินพุต A และ B ไปยัง GND และใช้ Push Button ต่อกับ VCC (+5V) เป็นอินพุต (หากยังไม่ได้ต่อจากการทดลองที่ 1.4)
2. รวมวงจรทั้งสองเข้าด้วยกัน
3. ใช้ LED ดวงที่ 1 แสดงค่า Sum
4. ใช้ LED ดวงที่ 2 แสดงค่า Carry
5. ทดลองอินพุตครบทั้ง 4 กรณี
6. ใช้ Oscilloscope 2 ช่อง Probe สัญญาณ Sum และ Carry พร้อมกัน — สังเกต Timing Relationship ระหว่าง Sum และ Carry
7. ใช้ Function Generator ป้อนสัญญาณ Square Wave ความถี่ 100 Hz ที่อินพุต A และใช้ Push Button ที่อินพุต B — ดู Waveform ของ Sum และ Carry บน Oscilloscope
8. บันทึกผลการทดลอง

#### ตารางที่ 1.5 บันทึกผลการทดลอง Half Adder

| A | B | Sum | Carry |
|---|---|---|---|
| 0 | 0 |     |       |
| 0 | 1 |     |       |
| 1 | 0 |     |       |
| 1 | 1 |     |       |

### คำถามท้ายการทดลองที่ 1.5

1. เพราะเหตุใด Half Adder จึงไม่สามารถบวกเลขหลายบิตได้
2. หากต้องการบวกเลข 2 บิตขึ้นไป จะต้องเพิ่มสัญญาณใด
3. วงจร Full Adder แตกต่างจาก Half Adder อย่างไร

## สรุปผลการทดลอง

อธิบายผลการทดลอง พร้อมวิเคราะห์ความถูกต้องของผลลัพธ์ และอธิบายสาเหตุของข้อผิดพลาด (ถ้ามี)

## คำถามท้ายใบงาน

1. เพราะเหตุใดการต่อวงจรบน Breadboard จึงต้องต่อ VCC และ GND ให้ถูกต้อง
2. หากอินพุตของ IC ถูกปล่อยให้เป็น Floating จะเกิดผลอย่างไร
3. วงจร XOR Gate สามารถสร้างจาก AND, OR, NOT Gate ได้อย่างไร จงเขียนสมการ
4. วงจร Half Adder ประกอบด้วย Logic Gate ใดบ้าง
5. Contact Bounce ที่สังเกตได้จาก Oscilloscope ส่งผลต่อการทำงานของวงจรดิจิทัลอย่างไร
6. การใช้ Oscilloscope ช่วยในการ Debug วงจรในแลปนี้อย่างไรบ้าง จงยกตัวอย่าง
